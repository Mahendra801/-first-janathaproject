Vector art is one of two art forms, used by computers, is another form of art bitmap (raster). Bitmap art is identified by file names ending in .gif, .bmp, .jpeg, .jpg, .png and .pcx. vector graphic files are usually stored as .eps, .ai or .cdr files.

Vector art is the mathematical algorithms created using computer software such as Adobe Illustrator, Freehand, Corel Draw, Quark and others. These programs use mathematical equations and geometric primitives (points, lines and shapes) to create art that is pure, the camera is ready, and can be scaled infinitely without loss of quality or clarity. The same technique can be used for business cards, posters and to a billboard.

The raster file (raster) dot matrix data structure uses pixels (small dots or colored squares) to create a complete picture.

The bitmap is technically characterized by the width and height in pixels, and the number of bits per pixel (color depth, which determines the number of colors that it can be).

Raster Graphics, such as photographs and graphic files created in Adobe Photoshop, Paint Shop Pro, and other programs for raster can be used for some applications, screen printing or digital printing. In most cases, especially with artists such as logos and numerous color printers require vector graphics to achieve the print quality that customer want and expect.
